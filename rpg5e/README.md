# RPG5E Logic Module

Comprehensive DND5E-style gameplay framework for your RPG prototype.

## Included Systems
- Core rules: abilities, modifiers, proficiency, XP and leveling.
- All base 5e classes metadata.
- Character model with checks, saves, attacks, rests, XP progression.
- Combat encounter loop with initiative and attack logs.
- Conditions support container on characters.
- Inventory and equipment system.
- Quest log and progression.
- Faction reputation tracking.
- Travel/day resolution with random encounter chance.

## Files
- `src/rules.js`
- `src/dice.js`
- `src/character.js`
- `src/combat.js`
- `src/systems.js`
- `src/index.js`
- `demo.js`

## Run Demo
```powershell
node rpg5e/demo.js
```
