# Settings and Options Reference

## Viewport
- `grid_size`: tile snap interval.
- `snap_to_grid`: enable/disable tile snapping.
- `show_collision`: overlay collision pass.
- `show_nav_overlay`: show nav/path overlays.
- `enable_realtime_preview`: live viewport updates.

## Project
- `autosave_seconds`: autosave interval.
- `autosave_enabled`: enable autosave.
- `backup_on_save`: keep backup revisions.
- `confirm_on_close`: prompt before closing with dirty state.

## User
- `enable_sound_preview`: preview SFX/music in editor.
- `compact_mode`: dense UI mode.
- `high_contrast_ui`: accessibility mode.
- `recent_projects[]`: recent project path list.

## Files
- `editor_settings.ini`: persisted settings.
- `autosave_project.db`: persisted project/asset DB.
- `autosave_tilemap.csv`: autosaved map grid.
- `autosave_manifest.json`: exported asset manifest.
